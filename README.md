## 📌 Overview

A **full-stack Task Manager application** that allows users to efficiently manage their daily tasks with authentication and CRUD operations.
Designed with a clean UI and scalable backend architecture.

---

## ✨ Features

* 🔐 Secure User Authentication (JWT based)
* 📝 Create, Read, Update, Delete tasks
* ✅ Mark tasks as completed
* 📊 Organized task listing
* 📱 Responsive UI design

---

## 🏗️ Architecture

Frontend and backend are separated for scalability:

```bash
task-manager-app/
 ├── frontend/   # React App
 ├── backend/    # Node + Express API
```

---

## 🛠️ Tech Stack

### 🔹 Frontend

* React.js
* HTML5, CSS3, JavaScript
* Axios

### 🔹 Backend

* Node.js
* Express.js
* JWT Authentication

### 🔹 Database

* MongoDB (Mongoose)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/rishipandey99/task-manager-app.git
cd task-manager-app
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm start
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## 🌐 API Endpoints

### 🔐 Auth

* POST /api/auth/register
* POST /api/auth/login

### 📋 Tasks

* GET /api/tasks
* POST /api/tasks
* PUT /api/tasks/:id
* DELETE /api/tasks/:id
